# Delta Java 8 pata Java 21

* Sintaxe Moderna (Redução de Boilerplate):
  - [ ] Var (Java 10+): Inferência de tipos local.
  - [ ] Records (Java 16+): Classes imutáveis de dados.
  - [ ] Text Blocks (Java 15): Strings multilinhas.
  - [ ] Switch Expressions (Java 14+): Mais conciso e seguro.
* Estrutura e Programação Funcional Avançada:
  - [ ] Sealed Classes (Java 17): Controle refinado sobre herança.
  - [ ] Pattern Matching (Java 16-21): Para instanceof e switch, eliminando casts manuais.
  - [ ] API de Coleções (Sequenced Collections): Novo acesso ordenado getFirst()/getLast().
* Concorrência e Performance (O salto principal):
  - [ ] Virtual Threads (Project Loom - Java 21): Threads leves que revolucionam I/O intenso, permitindo milhões de threads.
  - [ ] Garbage Collectors (G1/ZGC): Entender as melhorias de performance.
* Mudanças de Infraestrutura:
  - [ ] Módulos (JPMS - Java 9+): Entender a forte encapsulação.
  - [ ] Jakarta EE (javax. para jakarta.*):* Migração de namespaces (para quem usa Spring/Jakarta).
