# Functional Interface

- Possui apenas um método abstrato
- Runnable é um exemplo de Functional Interface
- Possui apenas o metodo run
- Uma Interface funcional é anotada com @FunctionalInterface

# Marker Interface 

- Interface que não possui métodos abstratos
- Exemplo: Serializable, 

# Predicate Interface

- Ajuda a testar nossos códigos individualmente
- Definida em java.util.function 
- É uma interface functional
- Seu metodo é boolean test(T t)
