# Statics

- Usada para declarar variáveis, métodos, blocos de classes internas e métodos static em interfaces

## Variable

- ela é alocada na memória apenas uma vez na classe
- ela então pertence a classe e não ao objeto

## Method

- pode ser acessado diretamente pelo nome e não precisa de nenhum objeto
- ele então pertence a classe e não ao objeto

## Block

- ele é um bloco de instrução dentro da classe que será executado quando a classe for carregada pela promeira vez na jvm
