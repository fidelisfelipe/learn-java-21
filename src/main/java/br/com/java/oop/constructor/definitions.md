# Constructor

- É um método especial e em uma classe que é chamado automaticamente quando um objeto de uma classe é inicializado
- Possui o mesmo nome de uma classe e é utilizado para inicializar o estado do objeto
- Tem o papel de atribuir valores iniciais aos atributos do objeto ou executar tarefas de configuração

## Características

- Nome e Assinatura
  - Possui o mesmo nome da classe
  - Nao possui retorno tipado, nem mesmo void, pois retorna o próprio objeto
  - Pode ou não ter parametros
  - Permite sobrecarga
  - São utilizados para inicializar os atributos ou as propriedades do objeto
