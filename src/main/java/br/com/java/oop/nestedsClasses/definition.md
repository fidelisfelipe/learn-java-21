# Nested Classes

- Precisamos definir uma classe dentro da outra pois isso temos as classes aninhadas
- O escopo de uma classe aninhada é limitado pelo escopo da classe que a envolve
- Ela é um membro da classe externa
- Existem alguns tipos de classes aninhadas

## Type's Nested Classes
  
- Static Nested Classes
  - Não podem acessar membros não estáticos da classe externa

- Non-static(Inner) Nested Classes
  - Local Classes
  - Anonymous Classes
