# Packages

- package é uma keyword
- deve ser definida no inicio do arquivo
- apenas uma declaração por arquivo
- se for omitida, a classe pertencerá ao pacote padrão
- o nome do pacote deve ser hierarquico e separado por pontos
- cada ponto representa um sub pacote

## Type of Packages

- Build-in Packages - java.util e java.lang
- User-defined Packages - pacotes do usuário

## Import

- Usado para importar pacotes incorporados e pacotes definidos pelo usuário para nossos arquivos java
- Podemos importar a classe específica
- Podemos importar o pacote completo, menos as classes dos sub-pacotes
- Todas as classes do pacote java.lang são importadas por default implicitamente.
- Pode ser importado métodos estáticos

## Confilct Names

- Atenção com classes do mesmo nome de pacotes diferentes como Date e Timer
